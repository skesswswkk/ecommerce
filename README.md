# 최저가 서비스 & 실시간 인기 상품 추천 서비스 제공 이커머스 

## 사용기술
- JAVA 17, Spring Boot 2.3, MyBatis, MySQL, Redis, Python, Locust
  
## 프로그램 주요 기능
- 회원
- 결제
  - 어댑터 패턴으로 외부 인터페이스 연동
- 배송
  - 스케쥴링 잡을 활용하여 상태 업데이트
- 검색
  - 캐싱 활용
- 카탈로그
  - NoSQL 활용하여 상품 정보 관리
  - Kafka 활용하여 카탈로그 변경 이벤트 발행 및 적용
- 주문
  - 외부 저장소 활용하여 작업 세션 관리
- 가격비교
